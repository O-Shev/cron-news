## Build and Run App
H2 database in memory is used to simplify.  
JRE 21 is required.

build the app:
```shell
mvn package
```

run the server:
```shell
java -jar .\server\target\server-0.0.1-SNAPSHOT.jar
```

run the client
```shell
java -jar .\client\target\client-1.0-SNAPSHOT.jar
```
